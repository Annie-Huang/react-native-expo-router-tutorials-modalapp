Modal can be created in one of the two ways:
- Using the 'Modal' component from React Native
- Using Expo Router's file-based syntax     << this tutorial will focus on this.

Syntax:
  <Stack.Screen
    name='modal'
    options={{ title: 'Modal', presentation: 'modal' }}
  />


Advantage to use file-based syntax to create the modal:
- We got dedicated URL for the page, good for SEO
